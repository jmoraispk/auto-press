How it works

Two halves. Use one or both.

The auto-clicker on its own already saves the day. Pair it with the phone bridge and you're not at the desk for any of it.

01 · Automate the laptop

Click the buttons you click a hundred times a day.

Capture Continue, Accept Change, Accept Edit once. CodeAway clicks them at the right moment, on the right monitor, at any DPI. Page Down to start, Page Down to stop.

  • Pixel-perfect template matching, scale-aware
  • Per-rule action: click, click + Enter, click + send text
  • Tray icon stays green while it scans
Demo · coming soon

02 · Remote control

Drive the agent from your phone, anywhere on your tailnet.

Install the PWA. See per-window status. Type your next reply, send it now or queue it for the next idle. Scroll the chat, snapshot the screen, all without opening the laptop.

  • Live status over Server-Sent Events
  • Send-now, queue, edit, delete — from the lockscreen
  • Scroll-and-screenshot to read the agent's history

Features

Everything you need to leave the keyboard.

A handful of small things, each polished. Nothing that asks for your email. Nothing that calls home.

Template-matching automation

Screenshot the button once. CodeAway clicks it whenever it appears — at any DPI, on any monitor, in any chat.

Phone bridge over Tailscale

Install the PWA on your phone. Status, send, scroll, screenshots — live, on your tailnet, no account.

Send and queue from anywhere

Type a reply on the phone. Sends now if the agent is idle, queues for the next idle if it's busy. Edit while it waits.

Local-first, telemetry-free

Everything runs on your machine. No cloud, no API keys, no analytics. Source is on GitHub — read the whole thing in 30 minutes.

Quick start

One command. Then you're in.

The installer drops uv, the source, and the deps into ~/codeaway. Nothing leaves your machine — read the script first if you'd like.
